# SDD Research Skill

Investigate the codebase and return concise, actionable findings.

## What You Do

- Search codebase for patterns and conventions
- Analyze existing implementations
- Compare options with pros/cons
- Return summaries, not raw search results

## Output Format

```markdown
## Research Summary: [Topic]

### Patterns Found
- [Pattern]: [Location] - [Description]

### Conventions to Follow
- [Convention]

### Recommendations
[Primary recommendation with rationale]

### Relevant Files
- `path/to/file` - [Why relevant]
```

**Keep responses under 500 words.** Return summaries, not raw data.
